MuscleLab to aplikacja wspierająca współpracę trenera personalnego oraz klienta. System pozwala zarządzać grafikiem, rezerwować treningi, kontrolować statusy spotkań oraz rozdzielać funkcje według roli użytkownika.

Status aplikacji

Backend Spring Boot
Security Basic Auth + BCrypt
Role CLIENT / COACH
Frontend HTML / CSS / JS

Klient

Przegląda trenerów, filtruje sloty, rezerwuje treningi i zarządza swoimi rezerwacjami.

Trener

Tworzy dostępne terminy, edytuje wolne sloty i obsługuje rezerwacje klientów.

Konto

Użytkownik może sam w pełni zarządzać swoim kontem oraz danymi na swoim koncie.
